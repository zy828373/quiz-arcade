### 1.4 配置文件

**config_team.yaml**（端口 8317）：
```yaml
host: ""
port: 8317
auth-dir: "./auths_team"
api-keys:
  - "team-api-key-1"
  - "team-api-key-2"
debug: true
proxy-url: "http://127.0.0.1:7897"
request-retry: 3
routing:
  strategy: "round-robin"
remote-management:
  allow-remote: true
  secret-key: "team-mgmt-key-2026"
```

**config_free.yaml**（端口 8318）：
```yaml
host: ""
port: 8318
auth-dir: "./auths_free"
api-keys:
  - "free-api-key-1"
  - "free-api-key-2"
debug: true
proxy-url: "http://127.0.0.1:7897"
request-retry: 3
routing:
  strategy: "round-robin"
remote-management:
  allow-remote: true
  secret-key: "free-mgmt-key-2026"
```

> ⚠️ secret-key 在首次启动后会被 CLIProxyAPI 自动替换为 bcrypt 哈希。如果忘记密码，需要手动改回明文再重启。

## Phase 2：Cloudflare 固定隧道穿透

### 2.1 安装 cloudflared
- winget 安装失败（源报错 0x8007139f）
- 改为手动下载：`cloudflared.exe` v2026.3.0，放在项目根目录

### 2.2 Quick Tunnel 失败
- 错误：`Error 1101: Worker threw exception`
- 原因：Cloudflare 服务端 `api.trycloudflare.com` 临时故障
- 决定：改用固定域名隧道

### 2.3 购买域名
- 平台：Cloudflare Registrar
- 域名：**codexapis.uk**
- 价格：$5.30/年（续费同价）
- 账户：Zhangyangzhang922@gmail.com

### 2.4 创建固定隧道
```powershell
# 登录认证
.\cloudflared.exe tunnel login
# → 浏览器授权 codexapis.uk
# → 证书保存到 C:\Users\AWSA\.cloudflared\cert.pem

# 创建隧道
.\cloudflared.exe tunnel create codex-pool
# → 隧道 ID: 632831ce-c805-4bd1-9239-985590c2aa61
# → 凭证: C:\Users\AWSA\.cloudflared\632831ce-...json

# DNS 路由
.\cloudflared.exe tunnel route dns codex-pool team-api.codexapis.uk
.\cloudflared.exe tunnel route dns codex-pool free-api.codexapis.uk
```

### 2.5 隧道配置文件
路径：`C:\Users\AWSA\.cloudflared\config.yml`
```yaml
tunnel: 632831ce-c805-4bd1-9239-985590c2aa61
credentials-file: C:\Users\AWSA\.cloudflared\632831ce-c805-4bd1-9239-985590c2aa61.json
ingress:
  - hostname: team-api.codexapis.uk
    service: http://localhost:8317
  - hostname: free-api.codexapis.uk
    service: http://localhost:8318
  - service: http_status:404
```

### 2.6 启动脚本
- `start_tunnel.bat` — 一个隧道同时服务 Team + Free
- 旧的 `start_tunnel_team.bat` 和 `start_tunnel_free.bat` 已删除

### 2.7 验证结果
- ✅ 隧道建立2个高可用连接（sjc01 + sjc10）
- ✅ 浏览器访问 `https://team-api.codexapis.uk/v1/models` 返回 `{"error":"Missing API key"}`（正常，说明隧道通了）
